### 특정 고객 거래내역 조회 서비스 API

#### 1. API 기능
1. 연도별 합계금액이 가장 많은 고객 조회
2. 연도별 거래가 없는 고객 조회
3. 연도별 관리점별 거래금액 합계 조회
4. 관리점 이관
5. 관리점 삭제
6. 입력한 지점의 거래금액 합계 조회

#### 2. 개발환경
- Spring Boot Framework
- Java 1.8.0_161
- Gradle
- Dependencies
    - Spring Boot DevTools
    - Lombok
    - Spring Configuration Processor
    - Spring Web Starter
    - Spring Data JPA
    - H2 Database
    - JDBC API
    - MyBatis
- IntelliJ IDEA
- Postman
